## Descripción del Proyecto  
**GestionZapateriaRiobamba** es una aplicación en **C#** para la gestión de ventas diarias de zapatos por marca (**Adidas, Nike, Puma**). La aplicación permite registrar, visualizar y filtrar ventas, calcular el total de ventas diarias y utilizar **GitHub** para el control de versiones y la colaboración en equipo.  
Este proyecto forma parte de una actividad de investigación formativa, con el objetivo de fortalecer habilidades técnicas y blandas en el desarrollo de software.  

---

## Funcionalidades Principales  
- **Registro de ventas:** Permite ingresar ventas diarias de zapatos por marca, incluyendo la cantidad vendida y el precio unitario.  
- **Visualización de ventas:** Muestra las ventas registradas en una interfaz gráfica amigable.  
- **Filtrado por marca:** Filtra las ventas según la marca seleccionada (Adidas, Nike, Puma).  
- **Cálculo del total de ventas:** Calcula automáticamente el total de ventas diarias.  
- **Control de versiones:** Utiliza GitHub para gestionar el código fuente, documentar cambios y colaborar en equipo.
